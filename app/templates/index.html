About the project

A translator interface, designed properly

AI Translator is a frontend concept built to show what a translation tool can feel like when interaction design gets the same attention as the underlying model. Every control here — the swap button, the character counter, the language badges — is built to answer one question instantly: what happens if I press this?

This build is intentionally backend-agnostic: drop it behind any translation API and the interface is ready to go.
